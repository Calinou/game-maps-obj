INTRODUCTION:
Sauerbraten (a.k.a. Cube 2) is a free multiplayer/singleplayer first person shooter, conceived, made and released by Wouter van Oortmerssen, available for download at: http://sauerbraten.org/. This map is intented as a little contribute to extend the original game, which can be improved thanks to the people willing to help.
--------------------------------------------------------------------------

Content:

1. "fc3.txt" (this file).
2. "fc3.ogz" ("Fantasy Castle" map for Sauerbraten).
3. "fc3.cfg" (map configuration file).
4. "fc3_wallpaper.jpg" (desktop wallpaper about this map)
5. "fc3.jpg" (screenshot of this map)

Instructions:

1. Browse through your "Sauerbraten" directory (usually: "C\Program Files\Sauerbraten").

2. Unzip "BA_fc3.zip", copy the "Packages" folder inside your "Sauerbraten" directory (usually: "C:\Program Files\Sauerbraten"). Press "Yes to all" when prompted to overwrite existing files/folder.

3. Run Sauerbraten!

4. Load the new map ("fc3.ogz"), by pressing "T" during the game, then type "/map fc3" and
press enter. For enjoying the gameplay at its best, It's suggested to change the gamma value
to 140 (press "T", then type "/gamma 140" and press enter. Default value is 100.
